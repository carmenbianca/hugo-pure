# hugo-pure

[English](https://github.com/undus5/hugo-pure/)|中文

一款简洁的 [Hugo](https://gohugo.io) 主题

## 特性

- 响应式设计 (适配移动设备)

- 暗色主题 (自动切换)

- 无 Javascript

- 支持多语言

## 安装

```
# 克隆本主题到你自己的项目
$ git submodule add https://github.com/undus5/hugo-pure themes/hugo-pure

# 将主题加入你的项目配置文件
$ echo "theme = 'hugo-pure'" >> config.toml

# 或者你可以直接启动演示项目
$ cd hugo-pure
$ ./run_demo.sh
```
